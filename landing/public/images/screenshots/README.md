## Especificaciones

- **Formato:** PNG o JPG
- **Resolución:** 1920x1080 (Full HD) o superior
- **Peso máximo:** 500KB por imagen
- **Optimización:** Usar herramientas como TinyPNG para comprimir

## Herramientas Recomendadas

- **Captura:** Windows Snipping Tool, ShareX, Lightshot
- **Edición:** Figma, Photoshop, GIMP
- **Optimización:** TinyPNG, ImageOptim
- **Anotaciones:** Skitch, Snagit

## Proceso de Captura

1. Abrir Aparca en modo incógnito (sin datos previos)
2. Usar datos de ejemplo consistentes
3. Capturar en resolución Full HD
4. Anotar elementos importantes si es necesario
5. Optimizar tamaño de archivo
6. Nombrar según la lista anterior
7. Colocar en esta carpeta
